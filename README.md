## Progetti

### Sito per polo di orientamento — Fabricamente
Next.js (App Router) · Payload CMS · TypeScript · Tailwind CSS · Vercel Postgres e Blob

Sito istituzionale per un polo di orientamento, online per un cliente reale
([polofabricamente.it](https://polofabricamente.it)). Contenuti gestiti da CMS (Payload), deploy su
Vercel. TypeScript strict, componenti decomposti e design system a token, test con Vitest.

### Gestionale Agenzia Viaggi
Java · Spring Boot · PostgreSQL

Applicazione gestionale completa: CRUD prenotazioni, pagamenti PayPal, verifica email, generazione
del PDF del biglietto con QR code. Architettura a livelli (controller, service, repository, DTO,
mapper con MapStruct), costruita da zero come esercizio e poi rifattorizzata con un'organizzazione
del codice più solida.

### Gestionale Autoscuola
Java · Spring Boot · Spring Modulith

Sistema multi-tenant per autoscuole, organizzato come modular monolith in moduli applicativi
distinti (Spring Modulith). Isolamento dei dati per scuola e controllo accessi RBAC con quattro
ruoli (amministratore, segreteria, istruttore, studente) gestito via Spring Security. Modello di
dominio e flussi progettati sulla base dell'esperienza diretta nel settore.

### CRUD API senza framework
Java · Javalin · PostgreSQL

API REST costruita senza Spring Boot per consolidare i fondamentali: routing, accesso al database,
gestione di richiesta e risposta, validazione e gestione degli errori.
